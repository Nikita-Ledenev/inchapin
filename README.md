# INCHAPIN

Демо-лендинг жилого комплекса бизнес-класса: два экрана, модальное окно с формой обратного звонка и адаптив под пять брейкпоинтов. Свёрстан по макету из Figma.

## Стек

- **Next.js 16** (App Router) + **React 19**
- **TypeScript** (strict)
- **SCSS-модули** — стили лежат рядом с каждым компонентом
- **next/font/local** — самохостинг шрифта Proxima Nova
- [react-select](https://react-select.com/) — выпадающий выбор квартиры
- [react-transition-group](https://reactcommunity.org/react-transition-group/) — монтирование и анимация модального окна
- [react-imask](https://imask.js.org/) — маска телефона
- [smooth-scrollbar](https://idiotwu.github.io/smooth-scrollbar/) — кастомный скролл страницы

## Требования

- **Node.js ≥ 20.9** (проект разрабатывался на 22 LTS)
- npm

Версия ноды зафиксирована в `.nvmrc`:

```bash
nvm use
```

## Запуск

```bash
npm install       # установка зависимостей
npm run dev       # дев-сервер на http://localhost:3000
```

Прод-сборка:

```bash
npm run build     # сборка
npm run start     # запуск собранного приложения
```

Линтер:

```bash
npm run lint
```

## Структура

```
app/
  layout.tsx           # корневой layout, метаданные, подключение шрифта
  page.tsx             # серверная страница-точка входа
  fonts.ts             # объявление Proxima Nova (300/400/600/700)
  globals.scss         # reset, базовая типографика, focus-стили
  styles/
    _variables.scss    # палитра-токены
    _mixins.scss       # миксины пяти брейкпоинтов
components/
  PageShell/           # клиентская оболочка: состояние модалки + композиция
  Header/  Logo/  MenuButton/  ApartmentSelect/  PhoneLink/  CallbackLink/
  Hero/
  About/  AboutBadge/  VideoTeaser/  VideoBadge/
  CallbackModal/  FormField/
  SmoothScroll/  HoverSwapText/
lib/
  cx.ts                # склейка классов
  useIsMounted.ts      # флаг монтирования для порталов
public/
  images/  icons/  video/
```

Компонент — это папка `Component/Component.tsx` + `Component.module.scss`. Директиву `"use client"` несут только компоненты, которым реально нужен браузер (скролл, порталы, состояние формы); `Hero` и `About` остаются серверными.

## Адаптив

Пять брейкпоинтов, опорный макет — 1440:

| Диапазон, px | Опорный макет |
| --- | --- |
| 0 – 767 | 360 |
| 768 – 1023 | 768 |
| 1024 – 1280 | 1024 |
| 1281 – 1769 | 1440 |
| 1770 + | 1920 |

- Первый экран занимает всю высоту окна (`100dvh`; на мобильном — `100svh`, чтобы не резалось панелями iOS).
- Второй экран имеет фиксированную высоту из макета.
- Шапка `fixed` и сжимается при скролле.
- С 768 текстовые контакты в шапке сворачиваются в круглую кнопку, на 360 бургер остаётся без подписи.

## Реализованные механики

- **Модальное окно** монтируется в DOM только когда открыто (`react-transition-group`), появляется через `opacity`, закрывается крестиком и по `Esc`, блокирует скролл страницы, ведёт фокус и возвращает его на триггер.
- **Форма**: floating-label, маска `+7 (999) 999-99-99`, лёгкая валидация обязательных полей с подсветкой ошибок; при успешной отправке данные выводятся в `console.log`.
- **Ховеры**: вертикальный текст-своп у кнопок и ссылок, разъезд линий бургера, `:active`-нажатия, анимированное подчёркивание телефона, пульсация у превью видео.
- **Видео о проекте** открывается на весь экран с автоплеем, по выходу из фуллскрина ставится на паузу и сбрасывается.
- **Кастомный скролл** на десктопе (smooth-scrollbar), с клавиатурной навигацией и корректным пиннингом `fixed`-шапки; на тач-устройствах и при `prefers-reduced-motion` — нативный скролл.
- Все анимации уважают `prefers-reduced-motion`, интерактивные элементы имеют состояние фокуса с клавиатуры.

## Ассеты

Изображения, иконки и лого — в `public/`, подключаются через `next/image`. Шрифт Proxima Nova (woff2) — в `app/fonts/`, подключён через `next/font/local` с самохостингом.
